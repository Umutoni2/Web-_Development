# HTML Basics

Simple project to understand the **structure and elements of an HTML5 webpage**.
This project is part of my self-learning journey to master HTML from scratch before moving to CSS and design.

---

## Files

* **base.html:** a basic valid HTML page with doctype and structure
* **practice.html:** a page used to practice headings, paragraphs, and lists
* **README.md:** this file — explaining the project and helping others learn

---

## What You’ll Learn

* How to write and organize an HTML5 document
* The meaning and use of tags like `<head>`, `<title>`, and `<body>`
* How to add text, headings, and lists
* The importance of using semantic elements for clean structure
* How to open and test HTML files in a browser

---

## Requirements

* No external libraries or CSS
* Must be **W3C valid**
* Each page should include:

  * `<header>` — for titles or top content
  * `<main>` — for main information
  * `<footer>` — for author or date
