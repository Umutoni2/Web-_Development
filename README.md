# What are we Able to Do Aside From Class Works

Simple self-made project to explore creativity and skills in **HTML and CSS**, built outside class assignments and intranet tasks.
This project helps me learn by doing, practice structure, and share coding basics with others.

---

## Files

* **index.html:** main webpage with sections, text, and images
* **about.html:** simple about page to describe my goals and learning journey
* **style.css:** stylesheet for colors, layout, and design
* **README.md:** this file — to guide and inspire other learners

---

## What You’ll Learn

* Structure of an HTML5 page (header, main, footer)
* How to link a CSS file to your HTML
* Basics of spacing, borders, and background color in CSS
* How to create and organize your first mini web project
* How to push your work to GitHub and write a clear README

---

## Requirements

* No external libraries or frameworks
* Must be **W3C valid**
* Each page should include:

  * `<header>` — for titles or navigation
  * `<main>` — for content
  * `<footer>` — for closing info or author name
